❓ HOW TO PLAY
🎲
ANSWER TO ROLL
Tap the dice to trigger a question. Answer correctly → roll normally (1–6 moves). Answer wrong → penalty of only 1 move this turn.
🏃
MOVE YOUR PLAYER
After rolling, all tiles you can reach glow green. Tap any glowing tile to walk there — the player moves step-by-step along the shortest path, spending one move per step.
⚑
REACH THE EXIT
Find the red flag and step onto it to escape. The maze is solved in as few total steps as possible.
🔄
REPEAT EACH TURN
Roll → move up to N steps → roll again. Hunters move after you exhaust your steps each turn, so plan carefully!
🔒
BREAKABLE WALLS
Green glowing 🔒 walls are shortcuts through the maze. Breaking one opens a new path — but you must earn it.
❓
ANSWER A QUESTION
Tap a 🔒 wall to trigger a question. Answer correctly → wall crumbles, path opens. Answer wrong → wall holds firm. Try again later.
⚙
AUTO PLACEMENT
By default, walls are placed automatically at the spots that save the most steps — so every breakable wall is a real shortcut.
✦
PICK YOUR OWN WALLS
Tap 🔒 WALLS in the game header to open the wall picker. Tap glowing tiles to mark them as breakable, then press START GAME. You can also set this in ⚙ EDIT QUESTIONS.
⛏
YOU HAVE 3 PICKAXES
Each game starts with 3 ⛏ pickaxes shown in the sidebar. They let you choose any wall to smash — not just pre-marked ones.
👆
HOW TO USE ONE
After using all your dice moves, tap USE (N) in the sidebar. Every wall that opens a real shortcut glows amber. Tap one to target it.
✅
ANSWER TO BREAK
A question appears. Answer correctly → pickaxe used, wall gone. Answer wrong → the pickaxe is NOT consumed, but you exit pick mode and must try next turn.
⚠
USE THEM WISELY
Pickaxes reset each new game but don't carry over. Save them for walls that cut off the hunter or open a direct route to the exit.
👻
GHOST — PHASE
The ghost passes through walls. It will always find you. Breakable walls won't stop it. Focus on the exit fast.
🧟
ZOMBIE — CHASER
Slow but persistent. Follows the shortest open path to you. Breaking walls can open shortcuts for it too — be careful.
👾
MONSTER — AMBUSHER
Predicts where you're heading and tries to cut you off. Changing direction throws it off — it adapts every few turns.
🚫
NO HUNTER
Relaxed mode. No enemy, no pressure — just solve the maze in as few steps as possible.
🧭
CLASSIC
You start top-left, exit is far away. Fixed positions every game. Good for learning the ropes.
💀
PUNISHMENT
Start and exit are placed randomly. You might begin far from the exit — or dangerously close to a hunter spawn. Use SET MODE → PUNISHMENT mid-game to reshuffle your position as a penalty.
✨
REWARD
Tap any valid tile on the maze to choose your own starting point. Use SET MODE → REWARD mid-game to relocate yourself as a strategic advantage.
✏️
CUSTOM QUESTIONS
Tap ⚙ EDIT QUESTIONS on the menu to write your own quiz questions for the breakable walls. Great for classrooms or trivia nights.
TIP: HUNTERS MOVE AFTER EVERY TURN — PLAN YOUR ROLLS CAREFULLY
